# 🛡️ AI Security Labs

## 🎯 What is this repo?

The repo contains AI Security Lab Simulation based on OWASP Top10 LLM! A collection of hands-on labs designed to teach and demonstrate common AI security vulnerabilities and defenses, following the OWASP Top 10 for Large Language Models framework.

## 📚 Available Labs

| Lab ID | Lab Name | Description | Status |
|--------|----------|-------------|--------|
| LLM01 | [Prompt Injection](LLM01-Prompt%20Injection/) | Learn about prompt injection attacks and defenses in AI systems | ✅ Available |

### How to Use
1. Click on any lab link above to navigate to the lab folder
2. Read the README.md in the lab folder for setup instructions
3. Follow the theoretical explanation in the markdown file
4. Run the lab and experiment with the concepts

## 🚀 Getting Started
Each lab is self-contained with its own setup instructions. Make sure you have the required prerequisites (Python, Node.js, etc.) before starting.
